# Sorting-Algorithms

It's a console app which implements few sorting algorithms written in C++

# Features:

  You can choose from the menu which algoritm do you want to determine.
  
  You can choose the way of selecting pivot: 
    1 - first element,
    2 - random element,
    3 - last element.
